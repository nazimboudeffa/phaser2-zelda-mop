Loading a spritesheet

To animate Link's sprite :

1- you need to reference a spritesheet

Example

game.load.spritesheet('link', 'assets/sprites/walking.tunic.png', 24, 32, 55);

2- create the animation

Example

player.animations.add('right', [0, 1, 2, 3, 4, 5, 6, 7], 16, false);

Animations contain a key that references the sprite sheet and a number of images
